# FickleFlight using Hasura's GraphQL APIs


### Run the app locally
```npm start```

### Build the app
```npm run build```


### Live demo
Note: Only these two routes are in the database 
  1. Singapore to San Francisco
  2. Singapore to Dubai
   
https://fickleflight-locofy-hasura.netlify.app/
